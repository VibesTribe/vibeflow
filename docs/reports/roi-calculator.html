ROI Calculator Workspace

Use this worksheet to reconcile mission telemetry with financial targets. Replace the placeholders with live data feeds once the telemetry exporters are wired in.

Inputs

  • Total tokens consumed per slice and agent
  • Unit economics (USD per 1K tokens / per minute)
  • Product metrics (activation, retention, conversion)

Outputs

  • Blended cost per mission and per slice
  • Estimated payback in days
  • Net contribution margin per release

Drop your existing spreadsheets or embed a Looker/Airtable dashboard here. The deploy pipeline copies this file directly to GitHub Pages so you can use it as a launch point for a richer calculator.
